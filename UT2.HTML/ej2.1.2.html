¿Qué es el DOM?

Una introducción al Modelo de Objetos del Documento

El Modelo de Objetos del Documento, más conocido por sus siglas DOM, es una de las tecnologías fundamentales en el desarrollo web. No es un lenguaje de programación, sino una interfaz que representa cómo el navegador ve un documento HTML. Como bien dice un desarrollador, es como un árbol de objetos que representa la página.

La organización Mozilla Developer Network (MDN) lo define de una forma más formal en su web:

El Modelo de Objetos del Documento (DOM) es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

Para interactuar con el DOM, usamos principalmente JavaScript. Por ejemplo, si quisiéramos cambiar el texto de un párrafo con el id miParrafo, podríamos usar un código como este: 

 
    // Seleccionamos el elemento por su ID
    const parrafo = document.getElementById("miParrafo");
    // Cambiamos su contenido de texto
    parrafo.textContent = "¡Texto cambiado con JavaScript!";

Entender el DOM es, en definitiva, el primer paso para crear páginas web dinámicas e interactivas.
