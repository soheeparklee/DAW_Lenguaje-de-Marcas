Tostada Perfecta con Aguacate


La receta definitiva para un desayuno rápido y nutritivo

La tostada con aguacate es, posiblemente, el desayuno más popular de la última década. Su simplicidad es su mayor fortaleza. Lo más importante es la calidad de los ingredientes.

No uses aguacates de mala calidad, eso es un error de novato. Este paso es fundamental.

Ingredientes:
Una rebanada de pan de masa madre
Medio aguacate maduro
Sal y pimienta al gusto
Opcional: un chorrito de aceite de oliva

  • Una rebanada de pan de masa madre
  • Medio aguacate maduro
  • Sal y pimienta al gusto
Opcional: un chorrito de aceite de oliva

Instrucciones:

  1. Tuesta el pan hasta que esté dorado.
    • No tuestes
    • La calidad
  2. Mientras, machaca el aguacate en un bol. La textura ideal no es un puré completo, sino que queden algunos trozos.
  3. Unta el aguacate sobre la tostada caliente.
  4. Añade sal y pimienta al gusto. No te pases, la OMS desaconseja el consumo excesivo de sal.
1. Tuesta el pan hasta que esté dorado.
2. Mientras, machaca el aguacate en un bol. La textura ideal no es un puré completo, sino que queden algunos trozos.
3. Unta el aguacate sobre la tostada caliente.
4. Añade sal y pimienta al gusto. No te pases, la OMS desaconseja el consumo excesivo de sal.

Para un toque gourmet, puedes añadir unas gotas de zumo de lima. El ácido ascórbico (C6H8O6) evitará que el aguacate se oxide.

¿Qué es el DOM?

Una introducción al Modelo de Objetos del Documento

El Modelo de Objetos del Documento, más conocido por sus siglas DOM, es una de las tecnologías fundamentales en el desarrollo web. No es un lenguaje de programación, sino una interfaz que representa cómo el navegador ve un documento HTML. Como bien dice un desarrollador, es como un árbol de objetos que representa la página.

La organización Mozilla Developer Network (MDN) lo define de una forma más formal en su web:

El Modelo de Objetos del Documento (DOM) es una interfaz de programación para los documentos HTML y XML. Facilita una representación estructurada del documento y define de qué manera los programas pueden acceder, al fin de modificar, tanto su estructura, estilo y contenido.

Para interactuar con el DOM, usamos principalmente JavaScript. Por ejemplo, si quisiéramos cambiar el texto de un párrafo con el id miParrafo, podríamos usar un código como este: 

            
                // Seleccionamos el elemento por su ID 
                const parrafo = document.getElementById("miParrafo"); 
                // Cambiamos su contenido de texto 
                parrafo.textContent = "¡Texto cambiado con JavaScript!";

Entender el DOM es, en definitiva, el primer paso para crear páginas web dinámicas e interactivas.
DOM (Document Object Model):
Es una interfaz de programación que representa el documento HTML como un árbol de objetos (nodos). Permite a JavaScript interactuar con la página web para crear, modificar o eliminar elementos, así como agregar eventos.
Nodos:
Son las unidades básicas que componen el árbol del DOM.
Árbol del DOM:
Los nodos están organizados jerárquicamente, formando una estructura de árbol. El nodo raíz es el propio documento, y los elementos HTML son los nodos que se van ramificando.
